What is QuickAdmin?
Have you ever experienced one of the following:
  • Explorer hangs when UAC is loading after I tried to run certain EXE that required administrator privileges.
  • UAC pops out and asks me did I want to allow the program to run... when I start a program that I'm already familiar.
  • Unable to run a program with elevated privileges in command line.
If you did, then QuickAdmin is the best solution for you.
It is a tool that designed to solve these kind of problems.
It has been tested countless of times, to confirm that it really works and stable.
It works on any Windows version, start from Windows Vista to 8.1, on 32-bit and 64-bit.
It uses .NET 2.0 and/or 4.0, included in Windows since Windows Vista.
And the most important thing is, it is a FREEWARE.
